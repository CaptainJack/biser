## Int format
```
1: 00             - FB             =      0 -           251
2: FC_00          - FC_FF          =    252 -           507
3: FD_00_00       - FD_FF_FF       =    508 -        66_043
4: FE_00_00_00    - FE_FF_FF_FF    = 66_044 -    16_843_259
5: FF_00_00_00_00 - FF_FF_FF_FF_FF =      0 - 4_294_967_295
```
